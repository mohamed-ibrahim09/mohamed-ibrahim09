## 🚀 Featured Projects

### 🖥️ DevOps System Monitor
> Real-time system metrics dashboard built with **Flask + psutil**, containerized with **Docker**

- Tracks CPU, memory, disk, and network usage via a live web interface
- Deployed as a Docker container for portable, reproducible environments
- Planned: Kubernetes deployment manifest + Prometheus/Grafana integration

### 🔄 CI/CD Pipeline — GitHub Actions
> Automated build, test, and deployment workflow for containerized applications

- Multi-stage pipeline: lint → test → Docker build → deploy
- Integrates with Docker Hub for image registry management
- Demonstrates zero-downtime deployment patterns

### 🔐 SecureLog — Cryptographic Web App
> Full-stack encryption platform implementing **Playfair, Vigenère, and AES+RSA hybrid** schemes

- **Stack:** Next.js frontend · Flask backend
- RSA upgraded to 512-bit keys with Miller-Rabin primality testing
- Blockchain-inspired tamper detection via chained hash logs

### ☁️ CloudForge — AWS Study & Docs Platform
> Personal AWS exam prep platform with deep-dive documentation and interactive study tools

- **Stack:** Next.js · Tailwind CSS — "Midnight Ember" dark design system
- Structured write-ups for EC2, IAM, S3, VPC with exam-focused notes
- Functions as both a study tool and a live portfolio artifact
